## 🔐 Tools & Utilities

### 🧩 [Decoder Tool](/decoder.html)
A handy browser-based utility to decode common formats used in security testing:
- JWT tokens
- Base64 (standard & URL-safe)
- URL encoding
- Hex
- and more...

Use it directly in your browser — no data is sent to any server.
